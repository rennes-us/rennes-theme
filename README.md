# rennes-theme

## Setup

 1. Install [themekit]
 2. Put `password`, `theme_id`, and `store` entries in config.yml
 3. Run `theme watch`

Additional tools:
 * <https://jshint.com>
 * <https://stylelint.io/>
 * <https://github.com/Dealerdirect/liquid-linter-cli> (not maintained though)

[themekit]: https://github.com/Shopify/themekit
